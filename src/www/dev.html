Seed Tool

This tool is a simplified version of the Ian Coleman BIP39 tool combined with the Samourai Wallet BIP47 tool. If accessing this tool online, NEVER enter a seed used to secure live funds.

For creating or interacting with seeds used with live funds, this tool MUST be used offline. Right click and 'Save Page As' to save a HTML file that can be opened in any web browser on an offline device or an amnesic operating system like Tails.

Usage

The most comomon use cases for this tool are:-

  • Generating a seed
  • Generating seeds via dice rolls or playing cards
  • Generating BIP85 child seeds (or check those generated by a hardware wallet)
  • Generating BIP47 payment codes and corresponding PayNym
  • Verifying wallet address generation from a given seed/passphrase combination
  • Verifying wallet entopy inputs (dice rolls or card picks)
  • Testing for a forgotten/incorrect passphrase (if you know a receive address)
